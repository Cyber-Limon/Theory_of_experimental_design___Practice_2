# Теория планирования эксперимента
# Практика 2

### Задание
1.	Пройти на СЭО опрос «Опрос 1 для практического занятия 2», указать список факторов (не менее 5), которые могут влиять на величину вашей будущей зарплаты;
2.	Дождаться появления опроса «Опрос 2 для практического занятия 2», проранжировать указанные в нем факторы, начиная от 1 – самый важный фактор и до k – наименее важный фактор (k – общее число факторов);
3.	Переписать результаты опроса 2 в таблицу – матрицу экспертного опроса и вычислить все величины по формулам 3.13-3.14/3.15 (см. Файл «К практическому занятию 2»);
4.	Проверить значимость коэффициента конкордации. В случае согласованности мнения экспертов построить диаграмму рангов, сделать выводы о необходимости исключения факторов из дальнейших экспериментов;
5.	В случае несогласованности мнения экспертов пересчитать коэффициент конкордации, убрав мнение одного из экспертов. (Подумайте, чье именно мнение следует убрать).
